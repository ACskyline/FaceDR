This is a OpenCV demo built with Qt Creator.
OpenCV version is 3.00
Qt Creator version is 3.6.1 Based on Qt 5.6.0(MSVC 2013, 32bit)

The function of this demo is to identify the most alike picture, from a large number of pictures, with the one you chose.
Therefore it can be used to match human faces, but the result may not be ideal.

It can only run on Windows and it is highly recommended to run it with a camera on your PC.
